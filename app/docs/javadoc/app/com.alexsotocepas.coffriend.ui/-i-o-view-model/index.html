open class IOViewModel

Classe base (superclasse) per als ViewModels que realitzen operacions d'entrada/sortida (I/O). Proporciona un mecanisme estandarditzat per emmagatzemar i actualitzar si una operació de xarxa o de base de dades ha estat satisfactòria. Aquesta informació es transmet de forma reactiva a les pantalles mitjançant un StateFlow.
