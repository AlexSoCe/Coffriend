User

data class User(val id: Int, val nom: String, val email: String, val rol: String = "client", val nivell: Int? = 0, val punts: Int? = 0, val idBotiga: Int? = null)

Representació de les dades de l'usuari autenticat a l'aplicació. Aquesta data class emmagatzema el perfil de l'usuari i el seu nivell de permisos (rol). La informació es manté de forma global mitjançant un objecte de companyia (companion object) per facilitar l'accés des de qualsevol punt de l'arquitectura.
